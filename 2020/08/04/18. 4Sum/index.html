18. 4Sum

source link

Description

Given an array nums of n integers and an integer target, are there elements a, b, c, and d in nums such that a + b + c + d = target? Find all unique quadruplets in the array which gives the sum of target.

Note:

The solution set must not contain duplicate quadruplets.

Signature
Analysis

The base idea is the same as previous problem 3sum , so here I just talk about some techniques and edge conditions.

  1. To avoid call malloc multiple times, estimate the memory size need to use and allocate it at one time.
  2. To avoid duplicate quadruplets, detect i-th, j-th element if equal to its previous one, and skip it if the condition is true.
